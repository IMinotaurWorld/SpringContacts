# SpringContacts

Есть два профиля. Default и init.
При выборе профиля init, будет происходить инициализация базы контактов из файла.
При выборе Default, база контактов будет создана с нуля.
В конфиге также присутствуют 2 пунтка настройки. initPath и savePath. В них нужно записать путь к файлу или файлам.

При запуске програма попросит ввести команду.
Чтобы посмотреть список доступных команд - введите Help.
